# blink.cmp

高速でバッテリー込みの Neovim 補完プラグイン。LSP・スニペット・バッファなどに対応し、タイポ耐性のあるファジー検索で候補を絞り込む。

設定ファイル: `config/nvim/lua/plugins/blink-cmp.lua`

---

## クイックリファレンス

| キー | 説明 |
|---|---|
| `<C-space>` | 補完メニューを表示 / ドキュメント表示切り替え |
| `<C-y>` | 候補を確定 |
| `<C-e>` | 補完メニューを閉じる |
| `<C-n>` / `<C-p>` | 次/前の候補を選択 |
| `<Tab>` | 候補を確定（スニペット中は次のプレースホルダへ）|
| `<S-Tab>` | スニペット前のプレースホルダへ |
| `<C-b>` / `<C-f>` | ドキュメントをスクロール（上/下） |

---

## 詳細

### 補完メニュー操作

- `<C-space>` — 補完メニューを表示、ドキュメントの表示/非表示を切り替え
- `<C-e>` — 補完メニューを閉じる
- `<C-y>` — 選択中の候補を確定
- `<C-n>` — 次の候補を選択
- `<C-p>` — 前の候補を選択
- `<Down>` — 次の候補を選択
- `<Up>` — 前の候補を選択

### ドキュメント操作

- `<C-b>` — ドキュメントを上にスクロール
- `<C-f>` — ドキュメントを下にスクロール

### スニペット操作

- `<Tab>` — 候補確定 / スニペット中は次のプレースホルダへ移動
- `<S-Tab>` — スニペット前のプレースホルダへ移動

### シグネチャヘルプ

- `<C-k>` — シグネチャヘルプの表示/非表示（デフォルト設定）

---

## 使い方

1. Insert モードで入力すると補完候補が自動表示される
2. `<C-n>` / `<C-p>` で候補を選択し、`<C-y>` または `<Tab>` で確定
3. スニペット展開後は `<Tab>` / `<S-Tab>` でプレースホルダ間を移動
4. `<C-space>` でドキュメントパネルを開き、`<C-b>` / `<C-f>` でスクロール

## 注意事項

- シグネチャヘルプは設定で `signature.enabled = true` を指定する必要がある（本設定では有効化済み）
- `<Tab>` はスニペット展開中かどうかで動作が変わる（`cmp.snippet_active()` で判定）
- ゴーストテキスト（インライン仮想テキスト）が有効になっており、確定前に補完プレビューが表示される
