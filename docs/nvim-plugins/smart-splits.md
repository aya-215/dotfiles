# smart-splits.nvim

左右上下の方向キーで直感的にウィンドウ移動・リサイズができるスプリット管理プラグイン。Tmux / WezTerm など外部マルチプレクサとのシームレスな統合にも対応。

設定ファイル: `config/nvim/lua/plugins/smart-splits.lua`

---

## クイックリファレンス

| キー | モード | 説明 |
|---|---|---|
| `<M-h>` | n/i/t | 左のウィンドウへ移動 |
| `<M-j>` | n/i/t | 下のウィンドウへ移動 |
| `<M-k>` | n/i/t | 上のウィンドウへ移動 |
| `<M-l>` | n/i/t | 右のウィンドウへ移動 |

---

## 詳細

### ウィンドウ移動（自分の設定）

- `<M-h>` — 左のウィンドウへ移動（ノーマル・インサート・ターミナルモード）
- `<M-j>` — 下のウィンドウへ移動（ノーマル・インサート・ターミナルモード）
- `<M-k>` — 上のウィンドウへ移動（ノーマル・インサート・ターミナルモード）
- `<M-l>` — 右のウィンドウへ移動（ノーマル・インサート・ターミナルモード）

### ウィンドウ移動（デフォルト推奨）

- `<C-h>` — 左のウィンドウへ移動
- `<C-j>` — 下のウィンドウへ移動
- `<C-k>` — 上のウィンドウへ移動
- `<C-l>` — 右のウィンドウへ移動
- `<C-\>` — 直前のウィンドウへ移動

### ウィンドウリサイズ（デフォルト推奨）

- `<A-h>` — 左に縮小
- `<A-j>` — 下に拡大
- `<A-k>` — 上に縮小
- `<A-l>` — 右に拡大

### バッファスワップ（デフォルト推奨）

- `<leader><leader>h` — バッファを左のウィンドウと交換
- `<leader><leader>j` — バッファを下のウィンドウと交換
- `<leader><leader>k` — バッファを上のウィンドウと交換
- `<leader><leader>l` — バッファを右のウィンドウと交換

---

## 注意事項

- プラグイン自体はキーマップを自動設定しない。ユーザー側で明示的に割り当てること
- `at_edge = 'stop'` を設定しているため、端のウィンドウで移動しようとしても外部マルチプレクサへ抜けない
- Tmux 連携を使う場合は `lazy = false` が必須
