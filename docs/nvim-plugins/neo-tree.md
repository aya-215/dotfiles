# neo-tree.nvim

ファイルシステムやバッファ、Git状態をツリー形式でブラウズできるサイドバープラグイン。

設定ファイル: `config/nvim/lua/plugins/neo-tree.lua`

---

## クイックリファレンス

| キー | 説明 |
|---|---|
| `<leader>E` | Neo-tree 開閉トグル |
| `<leader>ge` | Git 変更ファイル一覧を開く |
| `<leader>be` | バッファ一覧を開く |
| `<space>` | ノードの展開/折りたたみ |
| `<cr>` | ファイルを開く |
| `P` | プレビューモード切り替え（フローティング） |
| `H` | 隠しファイル表示切り替え |
| `/` | ファジーファインダー |
| `[g` / `]g` | 前/次の Git 変更ファイルへ移動 |
| `?` | ヘルプ表示 |

---

## 詳細

### ファイル操作

- `a` — 新規ファイル/ディレクトリ作成
- `A` — ディレクトリ作成
- `d` — ファイル/ディレクトリ削除
- `r` — 名前変更
- `c` — コピー
- `m` — 移動
- `y` — クリップボードにコピー
- `x` — クリップボードにカット
- `p` — クリップボードから貼り付け

### ファイルを開く

- `<cr>` — 開く
- `s` — 垂直分割で開く（※この設定では無効化）
- `S` — 水平分割で開く（※この設定では無効化）
- `t` — 新規タブで開く
- `w` — ウィンドウピッカーで開く

### ナビゲーション

- `<bs>` — ルートを1階層上へ
- `.` — 選択フォルダをルートに設定
- `C` — ノードを閉じる
- `z` — すべてのノードを折りたたむ
- `<` / `>` — 前/次のソースに切り替え

### フィルタリング・検索

- `/` — ファジーファインダー
- `D` — ディレクトリのみファジーファインダー
- `f` — フィルター入力
- `<C-x>` — フィルタークリア
- `H` — 隠しファイル表示切り替え

### プレビュー

- `P` — プレビューモード切り替え（フローティング）
- `l` — プレビューウィンドウにフォーカス
- `<C-f>` / `<C-b>` — プレビューをスクロール
- `<esc>` — プレビューモード終了

### ソート

- `o` — ソートメニュー表示
- `oc` — 作成日時でソート
- `om` — 更新日時でソート
- `on` — ファイル名でソート
- `os` — ファイルサイズでソート
- `og` — Git ステータスでソート

### その他

- `i` — ファイル詳細表示
- `R` — ファイルシステム再スキャン
- `#` — ファジーソーター

### コマンド

- `:Neotree` — Neo-tree を開く（デフォルト: ファイルシステム、左サイドバー）
- `:Neotree toggle` — 開閉トグル
- `:Neotree git_status` — Git 変更ファイル一覧
- `:Neotree buffers` — 開いているバッファ一覧
- `:Neotree float` — フローティングウィンドウで開く
- `:Neotree close` — 閉じる
- `:Neotree reveal` — 現在のファイルをツリーで表示

---

## 注意事項

- `s` / `S` キーは flash.nvim と競合するため無効化済み。分割で開くには `:split` / `:vsplit` を使うこと
- ドットファイルはデフォルト表示（`hide_dotfiles = false`）、Git ignore ファイルは非表示
- `.DS_Store` / `thumbs.db` / `desktop.ini` は常に非表示
