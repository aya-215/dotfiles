# lazygit でのマージ作業

## マージの基本操作

### ブランチをマージする

1. `3` キーでブランチビューに移動
2. マージしたいブランチにカーソルを合わせる
3. `M` キー（大文字）でそのブランチを現在のブランチにマージ

### マージコンフリクトの解決

コンフリクト発生時、以下の手順で解決：

1. ファイルビュー（`1`）でコンフリクトファイルを選択
2. `Enter` でステージングビューに入る
3. コンフリクト箇所で以下のキーを使用：

| キー | 動作 |
|------|------|
| `b` | 両方の変更を採用 |
| `←` | 左（ours）を選択 |
| `→` | 右（theirs）を選択 |
| `↑` / `↓` | 次/前のコンフリクト箇所へ移動 |

### マージの中止

マージ中に問題が発生した場合、`esc` または `M` で中止オプションが表示される。

## ヘルプ

lazygit 内で `?` キーを押すと、現在のビューで使えるキーバインドが表示される。
