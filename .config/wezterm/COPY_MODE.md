# WezTerm コピーモード設定まとめ

## コピーモード起動

| キー | 動作 |
|------|------|
| `Leader + y` (Ctrl+S → y) | コピーモード開始 |

---

## コピーモード内キーバインド

### 基本移動
| キー | 動作 |
|------|------|
| `h` | 左へ移動 |
| `j` | 下へ移動 |
| `k` | 上へ移動 |
| `l` | 右へ移動 |

### 単語移動
| キー | 動作 |
|------|------|
| `w` | 次の単語へ |
| `b` | 前の単語へ |
| `e` | 単語末尾へ |

### 行移動
| キー | 動作 |
|------|------|
| `0` | 行頭へ |
| `$` | 行末へ |
| `^` | 最初の非空白文字へ |

### ページ移動
| キー | 動作 |
|------|------|
| `g` | スクロールバック先頭へ |
| `G` | スクロールバック末尾へ |
| `Ctrl+u` | 半ページ上 |
| `Ctrl+d` | 半ページ下 |

### 画面内移動 (Vim H/M/L)
| キー | 動作 |
|------|------|
| `H` | 画面上部へ移動 |
| `M` | 画面中央へ移動 |
| `L` | 画面下部へ移動 |

### 文字ジャンプ (Vim f/t/F/T)
| キー | 動作 |
|------|------|
| `f` | 前方の指定文字へジャンプ |
| `F` | 後方の指定文字へジャンプ |
| `t` | 前方の指定文字の手前へ |
| `T` | 後方の指定文字の後ろへ |
| `;` | 同方向に再ジャンプ |
| `,` | 逆方向に再ジャンプ |

### 選択
| キー | 動作 |
|------|------|
| `v` | 文字選択モード |
| `V` | 行選択モード |

### ヤンク（コピー）
| キー | 動作 |
|------|------|
| `y` | 選択範囲をコピーしてモード終了 |

### 検索
| キー | 動作 |
|------|------|
| `/` | 検索開始 |
| `n` | 次のマッチへ |
| `N` | 前のマッチへ |

### モード終了
| キー | 動作 |
|------|------|
| `q` | コピーモード終了 |
| `Escape` | コピーモード終了 |

---

## Quick Select Mode

画面上のテキスト（URL、パス、Gitハッシュなど）にラベルを表示し、入力でコピーする機能。

### 起動
| キー | 動作 |
|------|------|
| `Leader + u` (Ctrl+S → u) | Quick Select Mode開始 |

### デフォルトでマッチするパターン
- URL (`https://example.com`)
- ファイルパス (`/path/to/file`)
- Gitハッシュ (`a1b2c3d`)
- IPアドレス (`192.168.1.1`)
- 数値

### 使い方
1. `Leader + u` で起動
2. ラベル（a, s, d, f...）が表示される
3. ラベルの文字を入力するとそのテキストがコピーされる
4. 大文字で入力するとコピー後に自動ペースト

## 検索モード内キーバインド

| キー | 動作 |
|------|------|
| `Enter` | 前のマッチへ |
| `Escape` | 検索終了 |
| `Ctrl+n` | 次のマッチ |
| `Ctrl+p` | 前のマッチ |
| `Ctrl+u` | パターンクリア |
